# 🎯 JavaScript 정규표현식 기초 (Basic Regex)

> 특정한 규칙을 가진 문자열의 집합을 표현하기 위해 사용하는 형식 언어입니다.

## 🔑 주요 기호
- `^`: 문자열의 시작
- `$`: 문자열의 끝
- `[]`: 대괄호 안의 문자 중 하나와 일치
- `+`: 앞의 문자가 1개 이상 반복됨

## 🛠️ 실전 예시: 0과 5만 포함된 문자열 검사
```javascript
const regex = /^[05]+$/;

regex.test("505");  // true
regex.test("501");  // false (1이 포함됨)
regex.test("");     // false (+는 최소 1개 필요)
```

## ⚠️ 주의사항

- `test()` 메서드는 불리언(true/false)을 반환하므로 조건문에 사용하기 적합합니다.
- 숫자를 검사할 때는 반드시 `String()`이나 `toString()`으로 형 변환을 먼저 해주어야 합니다.
