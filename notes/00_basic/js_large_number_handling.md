# 🔢 JavaScript에서 큰 숫자(Large Numbers) 다루기

> JavaScript의 기본 `Number` 타입은 표현할 수 있는 정수 범위에 한계가 있습니다.

## 🔑 범위의 한계
- `Number.MAX_SAFE_INTEGER`: 9,007,199,254,740,991 ($2^{53} - 1$)
- 이보다 큰 숫자를 다룰 때는 연산 결과가 부정확해질 수 있습니다.

## 🛠️ 해결 방법
1. **문자열 연산:** 숫자를 문자열로 받아 한 글자씩 처리합니다. (이번 문제 풀이 방식)
2. **BigInt:** 아주 큰 정수를 표현하기 위한 전용 자료형입니다. 숫자 뒤에 `n`을 붙여서 표현합니다.

```javascript
const big = BigInt("123456789012345678901234567890");
console.log(big % 9n); // BigInt 연산 가능
```

## ⚠️ 주의사항
- `Number`와 `BigInt`는 서로 섞어서 연산할 수 없습니다. (형 변환 필요)
- `Math` 객체의 메서드(예: `Math.max`)는 `BigInt`와 함께 사용할 수 없습니다.
