# ⚖️ 반복문(Iteration) vs 재귀(Recursion)

## 🔄 반복문 (While, For)
- **메모리:** 변수 값만 업데이트하므로 효율적.
- **안전성:** 무한 루프만 주의하면 스택 오버플로우 걱정이 없음.
- **용도:** 성능이 중요하거나 반복 횟수가 매우 많을 때.

## 🪞 재귀 (Recursion)
- **메모리:** 호출될 때마다 함수 실행 컨텍스트가 스택에 쌓임.
- **안전성:** 호출 깊이가 깊어지면 에러 발생 가능성 있음.
- **용도:** 복잡한 문제를 작은 단위로 쪼갤 수 있을 때 (예: 트리 구조 탐색, 팩토리얼).
